# Maze VR
Игрок оказывается в темном лабиринте с единственным предметом в руках - сканером местности.
Его главная задача - выбраться из мрачного лабиринта используя сканер и избегая столкновений с противниками.

## Платформы
- Windows, Linux, MacOS
